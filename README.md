# Still Moment - Meditation Timer

[![CI](https://github.com/stillmoment-app/stillmoment/actions/workflows/ci.yml/badge.svg)](https://github.com/stillmoment-app/stillmoment/actions/workflows/ci.yml)
[![License](https://img.shields.io/badge/license-Apache--2.0-blue.svg)](LICENSE)

A warmhearted meditation timer with Tibetan singing bowl gongs, guided meditation library, and warm earth tone design.

**Platforms**: iOS (SwiftUI) + Android (Jetpack Compose)
**Languages**: Deutsch, English
**Privacy**: 100% offline, no data collection

## Quick Start

```bash
# iOS
git clone https://github.com/stillmoment-app/stillmoment.git
cd stillmoment/ios && make setup
open StillMoment.xcodeproj

# Android
cd stillmoment/android
./gradlew build
```
