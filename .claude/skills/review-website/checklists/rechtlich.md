# Rechtlich

Rechtliche Anforderungen fuer deutsche Websites (TMG, DSGVO).

## Impressum (Pflicht)

`docs/impressum.html` muss enthalten:

### Pflichtangaben
- [ ] Vollstaendiger Name des Betreibers
- [ ] Anschrift (keine Postfach-Adresse)
- [ ] E-Mail-Adresse
- [ ] Telefonnummer (empfohlen)

### Erreichbarkeit
- [ ] Link im Footer sichtbar
- [ ] Maximal 2 Klicks von jeder Seite erreichbar

## Datenschutzerklaerung (Pflicht)

`docs/privacy.html` muss enthalten:

### Pflichtangaben
- [ ] Verantwortlicher (Name, Adresse, E-Mail)
- [ ] Art der erhobenen Daten (oder: keine Erhebung)
- [ ] Zweck der Datenverarbeitung
- [ ] Rechte der Nutzer (Auskunft, Loeschung, etc.)

### Bei GitHub Pages (minimal)
- Hinweis auf GitHub Pages Hosting
- Hinweis auf Server-Logs durch GitHub

### Erreichbarkeit
- [ ] Link im Footer sichtbar
- [ ] Maximal 2 Klicks von jeder Seite erreichbar

## Kontakt

### E-Mail
- [ ] Kontakt-E-Mail auf Support-Seite
- [ ] E-Mail ist erreichbar (keine Bounce)

**Aktuelle Kontakt-E-Mail:** `stillMoment@posteo.de`

## Footer-Links

Der Footer muss enthalten:
- [ ] Link zu Impressum
- [ ] Link zu Datenschutzerklaerung
- [ ] Link zu Support/Kontakt

## Melden wenn

- Impressum fehlt oder unvollstaendig
- Datenschutzerklaerung fehlt
- Links im Footer nicht funktionieren
- Kontakt-E-Mail nicht erreichbar
