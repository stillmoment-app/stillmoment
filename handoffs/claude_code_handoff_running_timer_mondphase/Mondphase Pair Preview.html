Mechanik · die zwei Formeln

Linear · Smoothstep

Schatten-Position

Linear interpoliert. Die letzten 10 % der Sitzung (cx −180 → −200) räumen die schwarze Scheibe komplett aus dem Bildausschnitt — kein dunkler Rest am Ende. 

// progress ∈ [0, 1]
cx = -progress * 200

// progress 0.0  → cx     0  (Neumond)
// progress 0.5  → cx  −100  (Halbmond)
// progress 0.9  → cx  −180  (Vollmond)
// progress 1.0  → cx  −200  (raus)

Halo-Intensität

Smoothstep — am Anfang fast unsichtbar, hält sich lang zurück, erblüht erst gegen Ende. Linear wäre zu aufdringlich und würde die Aufmerksamkeit von der Atmung wegziehen. 

eased = p * p * (3 - 2 * p)
α    = 0.02 + eased * 0.48

// α ∈ [0.02 … 0.50]
// p=0.25 → α ≈ 0.10
// p=0.50 → α ≈ 0.26
// p=0.75 → α ≈ 0.42
